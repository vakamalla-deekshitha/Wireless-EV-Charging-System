# Wireless-EV-Charging-System
Design and Development of a Rear-Mounted Inductive Wireless Charging System for Electric Vehicles. This project proposes an innovative rear-mounted coil architecture to improve wireless power transfer efficiency, and enhance charging safety compared to conventional bottom-mounted wireless EV charging systems.
Overview

This project presents a novel Rear-Mounted Inductive Wireless Charging System for Electric Vehicles (EVs). Unlike conventional wireless charging systems that place charging coils underneath the vehicle, the proposed design positions both transmitter and receiver coils at the rear side of the vehicle. This configuration reduces air-gap distance, improves magnetic coupling, minimizes eddy current losses, and enhances overall charging efficiency and safety.

Problem Statement

Traditional plug-in EV charging systems face several challenges:

Connector wear and maintenance issues
User inconvenience during charging
Electrical safety concerns
Exposure to environmental conditions

Existing wireless charging systems typically use bottom-mounted coils, which suffer from:

Large air gaps due to vehicle ground clearance
Coil misalignment issues
Reduced magnetic coupling efficiency
Increased eddy current losses
Heating of vehicle metallic components

This project aims to overcome these limitations through a rear-mounted coil arrangement.

Objectives
Improve wireless power transfer efficiency
